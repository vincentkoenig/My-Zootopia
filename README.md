# My Zootopia

A Python program that fetches animal data from the API-Ninjas API and generates a website displaying the results.

## Installation

1. Clone the repository
2. Install the dependencies:
```
   pip install -r requirements.txt
```
3. Create a `.env` file in the root directory and add your API key:
```
   API_KEY=your_api_key_here
```

## Usage

Run the following command:
```
python animals_web_generator.py
```
Then enter an animal name when prompted. The website will be generated as `animals.html`.
